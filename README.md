## ⚙️ Running the Portfolio Locally

This portfolio website is built using **React**, **TypeScript**, **Vite**, and **Tailwind CSS**.

### 1. Install Dependencies
```bash
npm install
```

### 2. Run in Development Mode
```bash
npm run dev
```

### 3. Build for Production
```bash
npm run build
```
